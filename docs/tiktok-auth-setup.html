Setting up TikTok Authentication

This guide will help you set up TikTok authentication for your HyperFlix application.

Prerequisites

  • A TikTok Developer account
  • Access to the TikTok Developer Portal
  • Your application's domain (e.g., hyper-flix.com)

TikTok Developer Portal Configuration

  1. Go to TikTok Developer Portal
  2. Navigate to your app settings

Basic Information

  • App Name: HyperFlix
  • Category: Business
  • Description: HyperFlix gives you the power to create viral marketing campaigns with AI-powered content generation and multi-platform distribution.
  • Platform: Web

URL Configuration

Add the following URLs to your app configuration:

  • Terms of Service URL: https://hyper-flix.com/terms
  • Privacy Policy URL: https://hyper-flix.com/privacy
  • Web/Desktop URL: https://hyper-flix.com

Credentials

Required Products

The following TikTok products are integrated with HyperFlix:

  • Login Kit
    • Enables secure user authentication
    • Required for other TikTok product integrations
    • Provides basic user profile information
  • Share Kit
    • Allows users to share content directly to TikTok
    • One-tap sharing functionality
    • Seamless integration with HyperFlix's content creation tools
  • Content Posting API
    • Enables direct video posting to TikTok
    • Supports both draft and published states
    • Requires Login Kit authentication
    • Perfect for automated content distribution

Required Scopes

The following scopes are required for HyperFlix functionality:

  • user.info.basic - Basic user information (display name and avatar)
  • video.list - Read-only access to user's public TikTok videos

Callback URLs

Add the following callback URLs to your app configuration:

Production:

https://hyper-flix.com/api/auth/callback/tiktok

Development:

http://localhost:3000/api/auth/callback/tiktok

Local Development Setup

  1. Update your .env.local file with the TikTok credentials:

    TIKTOK_CLIENT_KEY=awhpr1dpajln8v3l
TIKTOK_CLIENT_SECRET=UpPFhuCrwXXVPKgYRQbsIwrENDf3RlJV
  2. Add localhost:3000 to your TikTok app's allowed domains

Production Setup

  1. Add your production domain (hyper-flix.com) to the allowed domains in TikTok Developer Portal
  2. Update your production environment variables with the same credentials
  3. Ensure your site is served over HTTPS (required for TikTok Login Kit)

Testing

  1. Run the application locally
  2. Try signing in with TikTok
  3. Verify that the following data is received:
    • User's TikTok ID
    • Display name
    • Avatar URL

Troubleshooting

Common issues and solutions:

  1. Popup Blocked: Ensure popups are allowed for your domain
  2. Invalid Redirect URI: Double-check the callback URLs in TikTok Developer Portal
  3. Scope Not Authorized: Verify all required scopes are added in the app configuration

Security Considerations

  1. Never commit your TikTok credentials to version control
  2. Use environment variables for all sensitive information
  3. Implement proper CSRF protection (handled by NextAuth.js)
  4. Store tokens securely (handled by NextAuth.js)

Product Configuration

Share Kit Setup

  1. In the TikTok Developer Portal, enable Share Kit for your app
  2. Add the Share Kit SDK to your application: 
    <script src="https://sf16-website-login.neutral.ttwstatic.com/obj/tiktok_web_login_sdk/tiktok-sdk-v2.js"></script>
  3. Configure allowed domains for sharing in the Developer Portal

Content Posting API Setup

  1. Enable Content Posting API in your app's settings
  2. Request the necessary permissions:
    • Video creation and publishing
    • Draft management
    • Content status tracking
  3. Add the required endpoints to your API configuration

Important Notes:

  • Content Posting API requires business verification
  • Share Kit requires domain verification
  • All products must comply with TikTok's content guidelines
  • Rate limits apply to API endpoints
